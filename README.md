# 🌐 Portfolio Website with 3D Integration [Live Link](https://portfolio-beta-self-ocp8qcklm3.vercel.app/)

## 🚀 Overview
A responsive, single-page portfolio website showcasing personal profile, projects, and contact form — enhanced with a dynamic 3D model. Built as part of a web development internship assignment to demonstrate frontend skills and interactive design.

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS, Tailwind CSS**
- **JavaScript (Vanilla)**
- **Spline 3D Integration**

---

## 📂 Folder Structure

    /portfolio
    ├── /css
    │   └── styles.css
    ├── /js
    │   └── script.js
    ├── /components
    │   ├── hero.html
    │   ├── about.html
    │   ├── projects.html
    │   └── contact.html
    └── index.html

---

## 🌟 Features

- ✅ Fully responsive design for desktop and mobile
- ✅ Clean, modular code structure with comments
- ✅ Hero, About, Projects, and Contact sections
- ✅ 3D model integration for visual impact
- ✅ Form validation using regex and JavaScript
- ✅ Functional contact form using **EmailJS**

---

## 🧠 3D Integration

- **Method**: Spline (iframe embedded)
- **Location**: Hero section
- **Functionality**: Responsive animated 3D scene loaded via iframe
- **Link**: [Spline Scene](https://my.spline.design/holographicearthwithdynamiclines-TGQ2ykKSk61RmxG3BlGploUO/)

---

## 📩 Contact Form

- **Fields**: Name, Email, Message
- **Validation**:
  - Empty field checks
  - Email format using regex (`^[^\s@]+@[^\s@]+\.[^\s@]+$`)
